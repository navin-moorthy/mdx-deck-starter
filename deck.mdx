- Option + P -> Toggle Presenter Mode
- Option + O -> Toggle Overview Mode
- Option + G -> Toggle Grid Mode

### Importing single Built-in Theme

```jsx
import comic from "mdx-deck";
export const theme = themes.comic;
```

---

### Importing multiple Built-in Theme

```jsx
import { comic, prism } from "mdx-deck/themes";
export const themes = [comic, prism];
```
